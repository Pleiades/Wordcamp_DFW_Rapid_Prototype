=== Cookd Pro 3.2.0 Genesis Child Theme ===

Set up tutorials can be found at https://feastdesignco.com/tutorials


=== Installation Instructions ===

1. Upload the Cookd Pro theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Cookd Pro theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.
